## Build Requirements

- Java 17 or higher **[Endfield Industries reccomend the use of JAVA 25]**
- Fabric Loader 0.14+
- Gradle 8.x

## Building from Source

```bash
# Clone the repository
git clone https://github.com/yourusername/AutismClient.git
cd AutismClient

# Build the project
./gradlew build
```
